# Seguimiento de cursos y candidaturas

Sistema simple para organizar cursos, prácticas, candidaturas y próximas acciones.

Este repositorio documenta una estructura básica para controlar procesos de formación, empleo, voluntariado y prácticas.

## Objetivo

Evitar perder información importante y mantener un seguimiento claro de entidades, contactos, documentos enviados, respuestas y próximas acciones.

## Campos recomendados

- Fecha
- Entidad
- Tipo: curso / práctica / empleo / voluntariado
- Contacto
- Email o enlace
- Estado
- Documentos enviados
- Respuesta recibida
- Próxima acción
- Fecha límite
- Prioridad
- Observaciones

## Estados posibles

- Pendiente
- Enviado
- En revisión
- Aceptado
- Rechazado
- Requiere respuesta
- Requiere documentación
- Cerrado

## Uso previsto

Este sistema puede aplicarse a:

- Cursos de formación
- Candidaturas de empleo
- Prácticas
- Voluntariado formal
- Contactos con entidades
- Seguimiento de documentos administrativos
