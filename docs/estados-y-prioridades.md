# Estados y prioridades

## Estados recomendados

| Estado | Significado |
|---|---|
| Pendiente | Aún no se ha enviado o revisado |
| Enviado | La solicitud ya fue enviada |
| En revisión | La entidad está revisando |
| Aceptado | Resultado positivo |
| Rechazado | Resultado negativo |
| Requiere respuesta | Hay que contestar |
| Requiere documentación | Falta algún documento |
| Cerrado | No requiere más acción |

## Prioridades

| Prioridad | Uso |
|---|---|
| Alta | Plazo cercano, curso importante o posible empleo |
| Media | Seguimiento necesario, pero no urgente |
| Baja | Sin plazo inmediato |

## Revisión semanal

Cada semana revisar:

- respuestas pendientes
- fechas límite
- documentos faltantes
- próximas acciones
- prioridades
