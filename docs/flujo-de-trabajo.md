# Flujo de trabajo para seguimiento

## 1. Registrar oportunidad

Añadir la oportunidad a la tabla con fecha, entidad, tipo y enlace o contacto.

## 2. Preparar documentos

Comprobar si hacen falta:

- CV
- carta breve
- certificado
- justificante
- formulario
- portfolio

## 3. Enviar

Registrar fecha de envío y documentos enviados.

## 4. Revisar respuesta

Actualizar estado cuando haya respuesta.

## 5. Definir próxima acción

Ejemplos:

- responder email
- enviar documento
- pedir cita
- esperar resolución
- cerrar registro

## 6. Cierre

Marcar como cerrado cuando ya no haya acciones pendientes.
